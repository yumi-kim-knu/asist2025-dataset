This repository provides the dataset used in the ASIS&T 2025 paper titled "Exploring LLM AI in Automatic Generation of Abstracts for Research Publications."

* Dataset Contents:
asist_2024_paper_list.csv: A list of papers used for the study, including:
- paper_type (e.g., short, long, poster)
- no (paper number for reference)
- title (title of the original paper)

* Source:
Paper metadata was manually collected from the official ASIS&T 2024 Annual Meeting websites:
- Short & Long Papers: https://www.asist.org/meetings-events/am/am24/2024-annual-meeting-papers/
- Poster Papers: https://www.asist.org/meetings-events/am/am24/am24-posters/
